git init : tạo reponsitory trên máy local
git clone: lấy source code trên mạng về
git pull : đồng bộ từ trên mạng về máy
git add & git add . : thêm sửa file
git commit : thêm vào reponsitory
git push : đồng bộ từ máy lên reponsitory
git log :
git status : 
--> check xem trạng thái file 
git log -decorate -graph -oneline:
Các bước upload file lên reponsitory :
 + git clone https://github.com/thai-opps/CAKE-SHOP.git
 + Bước 1 : gõ lệnh "git status" : xem trạng thái file 
 + Bước 2: gõ lệnh  "git add ."
 + Bước 3: gõ lệnh " git commit -m "your comment": thêm vào reponsitory
 + Bước 4: gõ lệnh " git pull ": đồng bộ về máy
 + Bước 5: gõ lệnh " git push" : đẩy lển github
